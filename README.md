# KeyStroke_Logging_Demonstration

## 📌 Description
This is a simple Keylogger built using Python. It captures keyboard inputs and stores them in text and JSON formats. A GUI is created using Tkinter.

## 🚀 Features
- Records key presses, holds, and releases
- Saves logs in:
  - logs.txt (continuous keystrokes)
  - logs.json (structured format)
- Simple GUI with Start button

## 🛠️ Technologies Used
- Python
- Tkinter
- Pynput
- JSON

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python main.py

## 📂 Output Files
- logs.txt
- logs.json
